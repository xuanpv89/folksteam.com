CMS offline-ready: trang này không dùng CDN nên sẽ mở được ngay. Nó tạo file Markdown đúng schema blog hiện tại. Sau khi tải file, đặt vào thư mục tương ứng rồi commit/deploy.

Sau khi điền form thì làm gì?

  1. 1
    Bấm Download .md để tải file bài viết, hoặc Copy Markdown nếu muốn dán thủ công.
  2. 2
    Đặt file vào đúng đường dẫn đang hiển thị bên dưới form, ví dụ src/content/blog/en/test-1.md.
  3. 3
    Commit/deploy lại website. Khi build xong, bài sẽ tự hiện trên trang Blog và filter theo category.
